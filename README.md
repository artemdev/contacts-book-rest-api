# REST API for managing contacts with authentication

## Features
**Managing Contacts List**
  - Create contacts
  - Update contacts
  - Delete contacts

**User Authentication**
  - JWT-based authentication
  - Email confirmation

## Technologies
- **Python**
- **FastAPI**
- **Docker**
- **Poetry**
- **Pydantic**
- **SQLAlchemy**
- **Postgres**
- **Alembic**
- **Sphinx**

## How to Start a Project
1. Activate the Poetry shell:
```poetry shell```

2. Install dependencies
```poetry install```

4. Run the application:
```python3 main.py```
